# Peer Tutor Finder

Peer Tutor Finder is a web-based platform that connects students who need help in specific courses with peer tutors.

## Features

- Students can request help by course code
- Tutors can register by submitting a form
- Search tutors by course 
- PHP & MySQL backend
- Built with HTML, CSS, JavaScript, PHP, XAMPP, and MySQL

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## Setup Instructions

1. Clone the repository
2. Move the project to `htdocs`
3. Import the database using the provided `.sql` file
4. Database Configuration (the next paragraph explained this instruction)
5. Run Apache & MySQL using XAMPP

## Database Configuration

Each PHP file contains its own database connection.
Before running the project, you must update the database credentials inside every PHP file to match your local MySQL configuration 
(host, username, password, database name).

The project is intended to be run using XAMPP.
